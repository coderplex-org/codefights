In this meetup, you'll be forming pairs of two and competing against others to solve coding challenges in your favorite language. We'll use websites like Hacker Rank, Code Fights, Code Wars, Coder Byte and FreeCodeCamp's Algorithm Challenges. This will be done in the pair programming format. In case you're not familiar, pair programming is a coding technique in which two people work together on a single system. One, the driver, writes code while the other, the navigator, reviews it. The two programmers switch roles frequently.

The goal is to create an engaging competitive environment where everyone can challenge their minds by solving problems and competing against each other.
