In this meetup, you'll be forming teams and competing against others to solve coding challenges in JavaScript. We'll use websites like Hacker Rank, Code Fights, Code Wars, Coder Byte and FreeCodeCamp's Algorithm Challenges. This will be done in the pair programming format. In case you're not familiar, pair programming is a coding technique in which two people work together on a single system. One, the driver, writes code while the other, the navigator, reviews it. The two programmers switch roles frequently.

The goal is to create an comfortable collaborative environment where everyone can learn by competing against each other.
